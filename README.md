# React To-Do List App

An intuitive React application for managing daily tasks. Users can add, update, remove, and mark tasks as completed. Each task includes a title and detailed description, with an interactive toggle for viewing task details. 

## Features
- Add new tasks with title and description
- Update existing tasks
- Remove tasks
- Mark tasks as completed
- Toggle to view task details interactively


---


## Tech Stack
- React
- CSS(Responsive UI)
