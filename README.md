# React To-Do List App

A simple and intuitive To-Do List application built with Raect. This app allows users to create, update, remove and mark tasks as completed.
Each task includes a title and a detailed description. Users can also click on a task view its content and manage it interactively.

## Features
- Add new tasks with title and description
- Update existing tasks
- Remove tasks
- Mark tasks as completed
- Click to toggle task detail view
